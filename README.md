# 🧪 Rick and Morty App

Aplicação frontend desenvolvida com foco em consumo de API externa, utilizando a API pública de Rick and Morty para exibir personagens, episódios e localizações do universo da série.

---

## 🚀 Sobre o Projeto

A aplicação permite visualizar informações detalhadas dos personagens da série, como:

* Nome
* Status (vivo, morto ou desconhecido)
* Espécie
* Origem
* Episódios em que aparece

---

## 🛠️ Tecnologias Utilizadas

* React
* JavaScript
* CSS / Tailwind
* Axios 
* Vite

---

## 📡 API Utilizada

A aplicação consome dados da API pública:

* https://rickandmortyapi.com/

Principais endpoints utilizados:

* `/character`
* `/episode`
* `/location`

---

## 🎯 Funcionalidades

* 🔍 Listagem de personagens
* 🔎 Busca por nome
* 📄 Paginação de resultados
* 📌 Visualização de detalhes do personagem
* 🌍 Exibição de origem e localização
* 🎬 Lista de episódios relacionados

---

## ⚙️ Como Rodar o Projeto

### 1. Clone o repositório

```bash
git clone https://github.com/Felipe-Cordeiro-98/rick-and-morty-app.git
```

### 2. Acesse a pasta

```bash
cd rick-and-morty-app
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute o projeto

```bash
npm run dev
```

### 5. Acesse a aplicação
Abra o navegador e vá para:

```
http://localhost:5173
```
